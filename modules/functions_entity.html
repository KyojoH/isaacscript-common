getEntities

  • getEntities(matchingEntityType?: number, matchingVariant?: number, matchingSubType?: number, ignoreFriendly?: boolean): Entity[]
    • Defined in functions/entity.ts:137

    Helper function to get all of the entities in the room or all of the entities that match a specific entity type / variant / sub-type.

    Due to bugs with Isaac.FindInRadius(), this function uses Isaac.GetRoomEntities(), which is more expensive but also more robust. (If a matching entity type is provided, then Isaac.FindByType() will be used instead.)

    Example:

    // Make all of the entities in the room invisible
    for (const entity of getEntities()) {
      entity.Visible = false;
    }

    Parameters

    • Optional matchingEntityType: number

      Optional. If specified, will only return NPCs that match this entity type.

    • matchingVariant: number = -1

      Optional. If specified, will only return NPCs that match this variant.

    • matchingSubType: number = -1

      Optional. If specified, will only return NPCs that match this sub-type.

    • ignoreFriendly: boolean = false

      Optional. If set to true, it will exclude friendly NPCs from being returned. False by default. Will only be taken into account if matchingEntityType is specified.

    Returns Entity[]

    • Defined in functions/entity.ts:607

    Helper function to set the position of every entity in the room based on a map of positions. If an entity is found that does not have matching element in the provided map, then that entity will be skipped.

      The map providing the positions for every entity.
