getNPCs

  • getNPCs(): EntityNPC[]
    • Defined in functions/entity.ts:58

    Helper function to get all the NPCs in the room. Due to bugs with Isaac.FindInRadius(), this function uses Isaac.GetRoomEntities(), which is more expensive but also more robust.

    Example:

    // Remove all of the enemies in the room
for (const npc of getNPCs()) {
  npc.Remove();
}

    Returns EntityNPC[]
