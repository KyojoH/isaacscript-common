getRoomIndex

  • getRoomIndex(): int
    • Defined in functions/rooms.ts:23

    Helper function to get the room index of the current room. Use this instead of calling Game().GetLevel().GetCurrentRoomIndex() directly to avoid bugs with big rooms. (Big rooms can return the specific 1x1 quadrant that the player is in, which can break data structures that use the room index as an index.)

    Returns int
