fireProjectiles

  • fireProjectiles(npc: EntityNPC, position: Vector, velocity: Vector, projectilesMode?: ProjectilesMode, projectileParams?: ProjectileParams): EntityProjectile[]
    • Defined in functions/npc.ts:46

    Helper function to make an NPC fire a projectile. Returns the fired projectile. Use this function instead of EntityNPC.FireProjectiles(), since that returns void.

    Parameters

    • npc: EntityNPC

      The NPC to fire the projectile from.

    • position: Vector

      The staring position of the projectile.

    • velocity: Vector

      The starting velocity of the projectile.

    • projectilesMode: ProjectilesMode = ProjectilesMode.ONE_PROJECTILE

      The mode of the projectile. Optional. Equal to ProjectilesMode.ONE_PROJECTILE by default.

    • projectileParams: ProjectileParams = ...

      The parameters of the projectile. Optional. Equal to ProjectileParams() by default.

    Returns EntityProjectile[]

    The fired projectile.

isRaglingDeathPatch

  • isRaglingDeathPatch(npc: EntityNPC): boolean
    • Defined in functions/npc.ts:175

    Helper function to detect the custom death state of a Rag Man Ragling. When Rag Man Raglings die, they turn into a patch on the ground and can be revived by Rag Man at a later time. This causes them to show up as an "alive" enemy, so they should usually be filtered out of lists of alive enemies.

    Parameters

    • npc: EntityNPC

    Returns boolean
