temporarilyRemoveTrinket

  • temporarilyRemoveTrinket(player: EntityPlayer, trinketType: number): TrinketSituation | undefined
    • Defined in functions/trinketGive.ts:14

    Helper function to temporarily remove a specific kind of trinket from the player. Use this in combination with the giveTrinketBack function to take away and give back a trinket on the same frame. This function correctly handles multiple trinket slots and ensures that all copies of the trinket are removed, including smelted trinkets.

    Note that for simplicity, this function assumes that all smelted trinkets are non-golden.

    Parameters

    • player: EntityPlayer
    • trinketType: number

    Returns TrinketSituation | undefined

    Undefined if the player does not have the trinket, or TrinketSituation if they do.

temporarilyRemoveTrinkets

  • temporarilyRemoveTrinkets(player: EntityPlayer): TrinketSituation | undefined
    • Defined in functions/trinketGive.ts:58

    Helper function to temporarily removes a player's held trinkets, if any. This will not remove any smelted trinkets. Use this in combination with the giveTrinketBack function to take away and give back trinkets on the same frame.

    Parameters

    • player: EntityPlayer

    Returns TrinketSituation | undefined

    Undefined if the player does not have any trinkets, or TrinketSituation if they do.
