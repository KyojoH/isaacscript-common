setCollectibleSprite

  • setCollectibleSprite(collectible: EntityPickup, pngPath?: string): void
    • Defined in functions/collectibles.ts:349

    Helper function to change the sprite of a collectible pedestal entity.

    For more information about removing the collectible sprite, see the documentation for the "emptySprite" helper function.

    Parameters

    • collectible: EntityPickup

      The collectible whose sprite you want to modify.

    • Optional pngPath: string

      Optional. Equal to the spritesheet path to load (e.g. "gfx/items/collectibles/collectibles_001_thesadonion.png"). If not specified, will remove the sprite, making it appear like the collectible has already been taken by the player.

    Returns void
