getRandom

  • getRandom(seed?: number): float
    • Defined in functions/random.ts:13

    This returns a random float between 0 and 1. It is inclusive on the low end, but exclusive on the high end. (This is because RNG.RandomFloat() can return a value of 0.999, but it will never return a value of exactly 1.)

getRandomFloat

  • getRandomFloat(min: number, max: number, seed?: number): float
    • Defined in functions/random.ts:29

    This returns a random float between min and max. It is inclusive on the low end, but exclusive on the high end. (This is because RNG.RandomFloat() can return a value of 0.999, but it will never return a value of exactly 1.)

initRNG

  • initRNG(seed?: number): RNG
    • Defined in functions/random.ts:74

    Helper function to initialize an RNG object.

    Example:

    const startSeed = Game():GetSeeds():GetStartSeed();
    const rng = initRNG(startSeed);
    const fiftyFiftyChance = rng.RandomInt(2) === 0;

    Parameters

    • seed: number = ...

      The seed to initialize it with. (If you aren't initializing it with a seed, then don't use this function and instead simply call the RNG() constructor.)

    Returns RNG
