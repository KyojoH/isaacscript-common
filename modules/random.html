getRandom

  • getRandom(seed: int): float
    • Defined in random.ts:8

    This returns a random float between 0 and 1. It is inclusive on the low end, but exclusive on the high end. (This is because RNG.RandomFloat() can return a value of 0.999, but it will never return a value of exactly 1.)

getRandomFloat

  • getRandomFloat(min: int, max: int, seed: int): float
    • Defined in random.ts:24

    This returns a random float between min and max. It is inclusive on the low end, but exclusive on the high end. (This is because RNG.RandomFloat() can return a value of 0.999, but it will never return a value of exactly 1.)

    Example:

    const realNumberBetweenOneAndThree = getRandomFloat(1, 3, seed);
