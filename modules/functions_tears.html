getFireDelay

  • getFireDelay(tearsStat: float): float

    The EntityPlayer object stores a player's tear rate in the MaxFireDelay attribute. This is equivalent to how many tears the player can shoot per frame. If you already have a "tears" stat and you want to convert it back to MaxFireDelay, then use this function.
