deepCopy

  • deepCopy(oldTable: LuaTable, convertMapsToTables?: boolean, tableName?: AnyNotNil): LuaTable
    • Defined in functions/deepCopy.ts:12

    deepCopy recursively copies a table so that none of the nested references remain. It will refuse to copy tables that have metatables, since that indicates that it the table is a special object of some kind and cannot be copied in a generically safe way. The one exception is TypeScriptToLua maps, which this function has explicit logic to support.

    Parameters

    • oldTable: LuaTable

      The table to copy.

    • convertMapsToTables: boolean = false

      Whether or not to convert children Map elements to Lua tables. False by default.

    • tableName: AnyNotNil = ...

      Used to track the current key that we are operating on.

    Returns LuaTable
