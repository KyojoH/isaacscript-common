temporarilyRemoveTrinkets

  • temporarilyRemoveTrinkets(player: EntityPlayer, trinketType: TrinketType): TrinketSituation | null
    • Defined in functions/trinkets.ts:18

    Helper function to temporarily remove a trinket from the player. Use this in combination with the giveTrinketBack function to take away and give back a trinket on the same frame. This function correctly handles multiple trinket slots and ensures that all copies of the trinket are removed, including smelted trinkets.

    Note that for simplicity, this function assumes that all smelted trinkets are non-golden.

    Parameters

    • player: EntityPlayer
    • trinketType: TrinketType

    Returns TrinketSituation | null
